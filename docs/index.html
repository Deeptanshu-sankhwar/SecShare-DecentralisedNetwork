SecShare is a platform that enables clients connected to a server to enable message passing and file sharing among themselves and asynchronous downloading. This peer to peer sharing system enables quick file sharing over the network, making use of connected peers by splitting the requested file into multiple parts and concurrently driving it towards the requester. The asynchronous downloading system allows the user to download file quickly using the power of concurrency. The message system allows peers to communicate with each other. The whole peer to peer system is driven towards a decentralized model of network sharing. This project is a portion of the course Principles of Programming Language under Dr. Saurabh Joshi.
